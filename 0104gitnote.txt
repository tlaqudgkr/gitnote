*   git checkout . 커밋한 상태로 되돌아가기   *
git 에서의 체크아웃을 쓸경우 커밋한 상태로 되돌아가는 것을 말한다!
커밋 후 작업을 했는때 마음에 안들경우 git checkout . 을 하면 커밋한 상태로 되돌아간다. (작업한 것은 다 사라진다)
만약 중요한 작업을 끝낸 후 커밋하고 중요한 작업의 디자인을 바꿔보고 싶어서 바꿨는데 마음에 안들었을 경우 체크아웃기능을 사용해서 되돌아 갈 수있다.

원격 서버의 주소를 git에 등록하기    orgin 더하기!
git remote add origin /원격/서버/주소 

원격저장소에 반영하기!
git push origin master 를 사용하여 저장소에 반영하기!!!!

*     Tag 테그달기!     *

git checkout . 시 커밋한 번호 앞자리 6자리를 썼지만 tag를 달아 더 쉽고 간단하게 체크아웃 하기 위해서 tag를 달아준다!

git tag 테크할이름적고 커밋한번호 적어주기!
ex) git tag 0104 d7bd5c2a4c
